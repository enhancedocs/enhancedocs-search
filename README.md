## Installing

### Package manager

Using npm:

```bash
npm install enhancedocs-search
```

Using yarn:

```bash
yarn add enhancedocs-search
```

## Usage

```js
import EnhancedSearch from 'enhancedocs-search';

import 'enhancedocs-search/dist/style.css';

<EnhancedSearch accessToken="ed-abc123" {...props} />
```
